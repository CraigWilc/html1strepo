What are databases?

Databases are a way that we use to store usually large amounts of data all in one place which makes it easier to process the data inside and also find the data inside. Usually data bases are in tables and the data between them is linked.

  1. Oracle
  2. MySQL
  3. Microsoft SQL Server
  4. PostgreSQL
  5. MongoDB

A table of DBMS features

Feature Explanation
Data Integrity Using a DBMS allows for the ability to ensure your data is consistent and valid and hasn't been changed without authorisation
Data Security Using a DBMS allows for improved data security as it allows for user access control allowing the principle of least priviledge to be used preventing people who shouldn't have access to certain areas from getting into those area's
Back-ups Using a DBMS allows for easy back-ups as its all controlled from one central point it allows all the data to be backed up from one place.
